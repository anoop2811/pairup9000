# Pair Up 9000
To install the dependencies:

`npm install`

To run in dev mode:

`npm start`

Run the tests:

`npm test`

Build to 'dist/bundle.js':

`npm run dist`

Build and push to BlueMix/Cloud Foundry of your Choice:

`npm run dist && cf push mypairupboard -c "node server.js"`

thanks for listening.
